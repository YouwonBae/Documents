# git

## Clone

```
$ git clone https://gitlab.com/MAZE-dankook/self-driving-patrol-car.git
```

## Branch

### check branch

```
$ git branch -a
or
$ git branch -r
```

### branch update 

```
$ git remote update
```

### branch checkout

```
$ git checkout -t <branch name>
$ git branch -a
```

## Pull

### git pull to rocal

```
$ git pull
```

## commit

### check rocal status

```
$ git status
```

### commit to rocal

```
$ git add <file name>
or
$ git add .
```

### commit message

```
$ git commit -m <message content>
```

## Push

### git push to server

```
$ git push
```
